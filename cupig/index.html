Cupig's Key Collectathon

Cupig's Key Collectathon is a simple puzzle game where you collect keys to open treasure chests. Your speed is measured, you go for speed. I create this game to learn game creation from zero to Steam release.

I don't expect this game to be a success, I don't even think people will buy it. But it is perfect to learn the ropes, to learn the principles of game development and also to learn how Steam works.

Cupig's Key Collectathon has language support for several languages, Steam achievements and controller support.
